# Ocean of Madness

Вы принимаете командование пиратским судном, едва пережившим атаку легендарного Кракена. Пока команда латает пробоины, море начинает сводить вас с ума.

* **Ваша задача:** управлять матросами и устранять поломки в реальном времени.
* **Главное испытание:** когда наступает **Стадия Безумия**, логика мира выворачивается наизнанку. Чтобы починить брешь слева, вам придется отправить команду направо.
